Para el desarrollo de este trabajo de grado se desarrollaron una serie de complementos para la plataforma Moodle. Para poder hacer uso de ellos debera inicialmente tener instalada la plataforma. Para instalarla diríjase a la dirección http://download.moodle.org/ y descargue de preferencia, la versión estable mas reciente.

Proceda con el proceso de instalación correspondiente, según su sistema operativo y la versión de Moodle que desea instalar :
http://docs.moodle.org/26/en/Installing_Moodle

Una vez instalado y configurado, ya se esta en capacidad de instalar los nuevos complementos. A partir de la versión 2.5 existen 2 formas de hacerlo:

Administración del sitio → Extensiones → Instalar módulos externos
	
	Ahí se debe seleccionar el tipo  de complemento a instalar, y cargar el zip, en el formato 	estándar.

Copie la carpeta del complemento en la carpeta correspondiente al tipo de complemento que desea instalar, dentro del directorio raiz, asignele los permisos de archivo respectivos y finalmente desde la interfaz wb dirijase a  
		Administración del sitio → Notificaciones

	En el caso específico de este trabajo de grado se desarrollaron los siguientes tipos de 	complementos:
	
Bloques(blocks)
Filtro(filter)
Tema(theme)

Los plugins desarrollados estaran siendo continuamente actualizados en la cuenta de: gituhb.com/estarguars113
